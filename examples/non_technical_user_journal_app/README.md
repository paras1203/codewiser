# Example — Journal app (non-technical founder)

Demonstrates **micro-education**, **default proposer**, and **delegation detector**.

## Narrative

User: “I don’t understand hosting.” → Micro-ed one-liners before “cloud sync” probes.  
User: “Whatever you think” on stack → Delegation bulk sheet for infra + journaling prompts; `_meta.delegations[]` captures each accepted bullet.  
User: “Still don’t know” on typography → Default proposer offers friendly rounded pairing with calm palette + confirms.

Outcome: MVP blueprint lists local-first encrypted journal + nightly optional sync (Supabase free tier) explicitly tagged `defaults_applied` vs explicit user choices around privacy journaling.
