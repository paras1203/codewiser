# The People — extraction goals

**Phase ID:** `foundation/the_people`  
**Blueprint:** `personas`

## Must capture

| Field | Detail |
| ----- | ------ |
| `user_count_type` | `single_user` — one person’s private data \| `multi_user` — distinct users needing shared or cross-visible data (drives contradiction `storage_vs_multiuser` vs `constraints.storage_strategy`) |
| `roles[]` | Named archetypes / responsibilities |
| `tech_savviness` | low \| medium \| high |
| `usage_frequency` | daily \| weekly \| occasional |
| `primary_motivation` | Why they adopt |
| `current_frustrations` | concrete pains |
| `geo_focus` | Region/timezone relevance |
| `demographics_band` | coarse age/profession bucket |

Secondary personas repeat condensed subset.
