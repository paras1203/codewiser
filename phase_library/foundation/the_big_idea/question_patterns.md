# The Big Idea — question patterns

## PATTERN: story_capture

**priority_weight:** `{ "impact": 5, "dependency": 5, "uncertainty": 4 }`

**FORMAT:** Open text (+ micro-education only if domain jargon emerges)  
**ASK WHEN:** Phase entry  
**QUESTION:** "In plain words: what problem should disappear because this exists?"  
**SYNTHESIS:** distill pain → outcome bullet.

## PATTERN: scope_fence

**priority_weight:** `{ "impact": 3, "dependency": 3, "uncertainty": 3 }`

**FORMAT:** Confirm  
**ASK WHEN:** pitch length >120 words OR multi-domain signals  
**QUESTION:** "Here are 3 bullets I heard — number the one that's v1?"  
**SKIP WHEN:** minimalist static site.
